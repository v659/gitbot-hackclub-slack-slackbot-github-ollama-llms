🤖 What is GitBot?

GitBot is an AI-powered Slackbot built using Python, Slack Bolt, Ollama, and GitHub’s public API.

It supports live conversations, user personalities, multilingual translation, and GitHub integration.

⚙️ Slash Commands

  • /change-personality [name] – Set the bot's persona
  • /slack-me – Get your Slack user info privately
  • /slack-me-public – Post your Slack user info publicly
  • /translate [text] – Translate text to English
  • /trans-public [lang_code] [text] – Publicly translate a message

📦 Features

  • Responds to mentions using Ollama models
  • Integrates with the GitHub API to fetch top repos
  • Supports multi-language translation using Google Translate API
  • Changeable personalities via system prompts

🧠 LLM Models Used

  • Ollama: Chat models (e.g., teen-bot)
  • Google Translate: via Deep Translator

🚀 How to Run Locally Forever

nohup python3 main.py &

This keeps the bot running in the background and logs output to nohup.out.
